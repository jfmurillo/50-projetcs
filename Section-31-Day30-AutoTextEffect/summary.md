# Section 31 — Day 30: Auto Text Effect

**Course:** 50 Projects In 50 Days — HTML, CSS & JavaScript  
**Date completed:** 2026-05-12  

## What This Project Does

A typing animation where "We Are Coders" types itself character by character, resets, and repeats. A range slider controls the typing speed in real time.

## Key Concepts Learned

| Concept | Detail |
|---------|--------|
| `String.slice(0, n)` | Returns the first `n` characters of a string |
| Recursive `setTimeout` | Allows dynamic interval changes between calls |
| `input` event | Fires on every slider move |
| CSS border-right as cursor | Visual trick to mimic a blinking cursor |
| `300 / speed` | Inverse relationship: higher speed → smaller delay |
