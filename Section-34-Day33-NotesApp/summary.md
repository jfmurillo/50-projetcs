# Section 34 — Day 33: Notes App

**Course:** 50 Projects In 50 Days — HTML, CSS & JavaScript  
**Date completed:** 2026-05-12  

## What This Project Does

Sticky-note app with Markdown support. Users add, edit, delete, and persist notes via `localStorage`. Each note toggles between raw Markdown textarea and rendered HTML preview.

## Key Concepts Learned

| Concept | Detail |
|---------|--------|
| `document.createElement()` | Creates elements dynamically |
| `marked(string)` | Converts Markdown to HTML |
| `localStorage.setItem` | Persists data across reloads |
| `JSON.stringify / JSON.parse` | Serialises arrays for storage |
| Edit/Preview toggle | Show textarea or div based on mode |
