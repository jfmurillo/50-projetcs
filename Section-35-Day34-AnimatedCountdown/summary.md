# Section 35 — Day 34: Animated Countdown

**Course:** 50 Projects In 50 Days — HTML, CSS & JavaScript  
**Date completed:** 2026-05-12  

## What This Project Does

Real-time countdown to New Year's Day. Shows days, hours, minutes, seconds. When it hits zero, a congratulatory message slides in. Transitions use CSS keyframe animations.

## Key Concepts Learned

| Concept | Detail |
|---------|--------|
| `new Date(string)` | Parses a date string into a Date object |
| Date subtraction | Returns difference in milliseconds |
| Modulo chaining | Extracts hours from leftover-after-days, etc. |
| `setInterval(fn, 1000)` | Calls `fn` every second |
| `animation-fill-mode: forwards` | Keeps final animation frame after it ends |
