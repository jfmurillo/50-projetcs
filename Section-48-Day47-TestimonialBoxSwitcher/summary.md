# Section 48: Day 47 - Testimonial Box Switcher

## Project Overview
A testimonial carousel that automatically cycles through user testimonials with smooth fade transitions.

## Key Concepts
- CSS animations and fade transitions
- JavaScript setInterval for auto-switching
- DOM manipulation to update quote, image, username, and role
- Array iteration and index cycling with modulo operator

## Implementation
- Testimonials stored as array of objects (quote, name, role, photo)
- Auto-switch every 3 seconds using setInterval
- CSS opacity and transform transitions for smooth fade effect
- updateTestimonial() function cycles through the array

## Files
- index.html - Testimonial card structure
- style.css - Card layout and fade animation styles
- script.js - Switcher logic with auto-rotation
