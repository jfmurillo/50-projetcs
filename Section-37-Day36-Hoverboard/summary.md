# Section 37 — Day 36: Hoverboard

**Course:** 50 Projects In 50 Days — HTML, CSS & JavaScript  
**Date completed:** 2026-05-12  

## What This Project Does

500 squares that light up with a random colour + glow on hover, then slowly fade back. Uses dual CSS transition timing: instant on hover, slow fade-out on leave.

## Key Concepts Learned

| Concept | Detail |
|---------|--------|
| DOM creation loop | `createElement` × N, append each |
| `e.target` | References the specific element that fired the event |
| `style.boxShadow` | Template literal to inject colour into glow |
| Dual CSS transition trick | `:hover { transition: 0s }` vs default slow transition |
| `mouseover` / `mouseout` | Hover enter/leave events |
